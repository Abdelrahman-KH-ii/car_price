# CarValue — Used Car Price Predictor

A Django web app that predicts used car prices using an XGBoost ML model.

## Quick Start

```bash
git clone https://github.com/Abdelrahman-KH-ii/car_price.git
cd car_price
docker compose up --build
```
Open → **http://localhost:8000**

## Tech Stack
Django · XGBoost · PostgreSQL · Docker · Nginx · Gunicorn · WhiteNoise

## Features
- User registration & login
- 27 brands / 150+ models with auto-fill specs
- Price prediction in INR (Lakh) + USD equivalent
- 37 automated tests
